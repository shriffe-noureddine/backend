Team Project Steps

Day 1 - Morning - Planning your project

  • Read the specifications
  • Analyze needs (together)
  • Distinguish the different components of the project (together)
  • Break down the project into different tasks (together)
  • Estimate the time for each task
  • Allocate tasks
  • Create the Conceptual Data Model (CDM) with DBDesigner or MySQLWorkbench or other
  • Zoning (models, mockups) pages (already done, to save time)
  • Organize your code (create files and directories from the beginning)
  • Use the tools necessary for teamwork (Git, file sharing, etc.)

Day 1 - Morning - Starting the code

  • Start with the backend part (without any style, just the features)
  • Use AJAX to make auto-completion works on search
  • Use SQL joins (INNER JOIN)
  • Use SQL subqueries (if necessary)

Day 2 - Finishing Backend & Starting frontend

  • Start the frontend part
  • Finish the backend part before 12pm!!!!!!
  • Manage priorities well (functionalities before cosmetics)
